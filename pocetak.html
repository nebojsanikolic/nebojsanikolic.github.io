React

*Ovaj vodič ne podrazumeva nikakva postojeća znanja o React-u.

1. Dodajte React na vašu stranicu


Korak 1
Dodajte DOM kontejner u HTML.

Prvo otvorite HTML stranicu koju želite da uredite. Dodajte praznu oznaku <div> da biste označili mesto na kojem želite da prikažete nešto pomoću React-a. Na primer:

<!-- ... existing HTML ... -->
<div id="like_button_container"></div>
<!-- ... existing HTML ... -->

Savet
Dali smo ovom <div>-u jedinstveni atribut id HTML. To će nam omogućiti da ga kasnije pronađemo iz JavaScript koda i u njemu prikažemo komponentu React.

Korak 2
Dodajte oznake skripte.

Zatim dodajte tri <script> oznake na HTML stranicu neposredno pre završne </body> oznake:

<!-- ... other HTML ... -->
<!-- Load React. -->
<!-- Note: when deploying, replace "development.js" with "production.min.js". -->
<script src="https://unpkg.com/react@17/umd/react.development.js" crossorigin></script>
<script src="https://unpkg.com/react-dom@17/umd/react-dom.development.js" crossorigin></script>

<!-- Load our React component. -->
<script src="like_button.js">

Prve dve oznake učitavaju React. Treći će učitati kod vaše komponente.

Korak 3
Napravite React komponentu.

Napravite datoteku pod nazivom like_button.js pored vaše HTML stranice.

Otvorite ovaj početni kod i nalepite ga u datoteku koju ste kreirali.

Savet
Ovaj kod definiše React komponentu koja se zove LikeButton. Ne brinite ako to još uvek ne razumete - kasnije ćemo pokriti blokove React-a u našem praktičnom vodiču i glavnom vodiču za koncepte. Za sada, samo da se prikazuje na ekranu!

Nakon početnog koda, dodajte dva reda na dno like_button.js:

// ... the starter code you pasted ...

const domContainer = document.querySelector('#like_button_container');
ReactDOM.render(e(LikeButton), domContainer);

Ova dva retka koda pronalaze <div> koji smo dodali u naš HTML u prvom koraku, a zatim u njemu prikazuju naše dugme „Sviđa mi se“ Reaguj komponentu.

To je to!

Ne postoji četvrti korak. Upravo ste dodali prvu komponentu React na svoju veb lokaciju.
